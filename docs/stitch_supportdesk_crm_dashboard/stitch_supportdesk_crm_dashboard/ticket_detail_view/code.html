TKT-001

API Integration failure on checkout endpoint

Created Jun 2, 2025 · Last updated 3 hours ago · customer@email.com

Description

The checkout endpoint `/api/v1/checkout` is returning a 500 Internal Server Error for users trying to process payments with international credit cards. This was first reported by several enterprise customers in the EMEA region.

Steps to reproduce:

  • Set account locale to 'FR' or 'DE'
  • Attempt checkout with a test Visa card
  • Observe 500 status code in network logs
